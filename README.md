# 🖼️ Sheet Image Generator

Sheet Image Generator is a modern web application built with **Next.js**, **React**, and **TypeScript** that automates the process of generating personalized images from data stored in Google Sheets. Instead of manually creating images for each individual, users can simply provide a Google Sheets URL, fetch the spreadsheet data, and generate customized images in a fast and efficient workflow.

The application securely communicates with the **Google Sheets API** through **Next.js API Routes**, ensuring that API keys remain protected while eliminating the need for a separate backend server. It validates the provided spreadsheet, retrieves the records, and displays them in an organized interface where users can preview and manage the data before generating images.

Designed with performance and usability in mind, the project features a clean, responsive interface and a streamlined workflow for bulk image generation. It is suitable for creating certificates, ID cards, event passes, invitations, promotional banners, greeting cards, and other personalized graphics using spreadsheet data.

## ✨ Features

- 📄 Fetch data directly from a Google Sheets URL.
- ✅ Validate spreadsheet URLs before processing.
- 👥 Display records in a clean and organized interface.
- 🖼️ Generate personalized images dynamically from spreadsheet data.
- ⚡ Fast and responsive user experience.
- 🔒 Secure API communication using Next.js API Routes.
- ☁️ Easy deployment on Vercel without a separate backend.
